This is a Bun-based CLI that adds deployment, editor, pre-commit, and API client scaffolding to supported React projects.

Use Bun for local development and scripts.

Common commands:

```bash
bun run build
bun run typecheck
bun test
```

For product behavior, supported frameworks, and generated outputs, see [README.md](./README.md).
For implementation details and the CLI flow, see [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md).
